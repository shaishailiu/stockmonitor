# 每日报告格式说明

## 报告结构

每日15:30扫描后，会发送以下完整报告：

### 1. 每个预警的详细消息（按级别排序）

**格式：**
```
🔴/🟠/🟡 [资产名称](富途链接) 触发底部信号

🎯 触发原因：
  • 具体触发条件

📉 价格信息：
  • 当前价：XXX
  • 52周高点：XXX (日期)
  • 回撤幅度：XX%

📈 技术指标：
  • RSI(14)：XX | 周线RSI：XX
  • 成交量：XX%（相对60日均量）

⚠️  基本面检查：
  ✅ 未发现明显风险
  或
  ❌ 存在风险因素：...

💯 综合评分：XX/100

💡 [具体操作建议]

⏰ 时间戳
```

**每个预警独立发送**，便于点击链接查看详情。

---

### 2. 汇总报告

**格式：**
```
============================================================
📊 投资监控完整扫描报告 - 2026-XX-XX HH:MM
============================================================

扫描资产：22只（5只A股 + 12只港股 + 5只美股）
数据源：腾讯财经（A股+港股完整52周）+ Alpha Vantage（美股100天）

============================================================
📊 投资监控日报 - 2026-XX-XX
============================================================

🔴 红色预警 (X个)：
  • 资产名 - 回撤XX% - 评分XX

🟠 橙色预警 (X个)：
  • 资产名 - 回撤XX% - 评分XX

🟡 黄色预警 (X个)：
  • 资产名 - 回撤XX% - 评分XX

共触发 X 个预警信号
============================================================
```

---

### 3. 无预警时的报告

**格式：**
```
📊 投资监控日报 - 2026-XX-XX

✅ 今日扫描完成，暂无新预警触发

扫描资产：22只（5只A股 + 12只港股 + 5只美股）
数据源：腾讯财经 + Alpha Vantage

⏰ 2026-XX-XX HH:MM:SS
```

---

## 消息发送规则

1. **有预警时：**
   - 先发送每个预警的详细消息（1条/资产）
   - 最后发送汇总报告（1条）
   - 红色预警置顶，然后橙色，最后黄色

2. **无预警时：**
   - 仅发送1条简短日报

3. **消息间隔：**
   - 每条消息间隔2秒，避免刷屏
