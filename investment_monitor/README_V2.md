## 🚨 预警体系

### 🟡 黄色预警（进入观察）
- 回撤 ≥ 30% + RSI(14) < 35
- 回撤 ≥ 30% + 估值 < 历史30%分位

**操作建议：** 观察，暂不操作

### 🟠 橙色预警（持续关注）
- 回撤 ≥ 40% + RSI(14) < 30 + 周线RSI < 35
- 回撤 ≥ 40% + 估值 < 历史15%分位
- 回撤 ≥ 40% + 成交量 < 60日均量的50%

**操作建议：** 可小仓位试探（10%）

### 🔴 红色预警（重点关注）
- 回撤 ≥ 50% + 周线RSI < 30
- 回撤 ≥ 50% + 估值 < 历史10%分位
- 回撤 ≥ 50% + MACD周线底背离

**操作建议：** 可分批建仓（20-30%）
