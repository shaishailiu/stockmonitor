## 推送输出中链接丢失问题（3月10日晚）

### 用户反映：今天下午推送的股票报告里，股票名称的链接丢了
- 代码本身没问题：`make_link()` 能正确生成 `[名称](url)` Markdown 格式
- 测试确认：`python3` 直接调用 `make_link('美团', item)` 输出 `[美团](https://www.google.com/finance/...)` 完全正常
- **根本原因：cron 任务的 AI 在整理脚本输出时，把 Markdown 链接 URL 部分去掉了**
  - 原始输出应为：`🟠 橙色预警 [03690（美团）](https://...) 触发底部信号`
  - AI 复述时变成：`🟠 橙色预警 [03690（美团）] 触发底部信号`
- **修复方案**：在任务 prompt 里明确强调保留 Markdown 链接格式（`[名称](url)` 不得拆分）
- **已于 21:07 更新任务 prompt**，新增强调：`[名称](url)` 格式必须完整保留，不得只保留 `[名称]` 而丢弃 `(url)` 部分
- 明天 16:30 生效验证
